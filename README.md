# 🎬 Creep Archives

Projeto de horror interativo desenvolvido com HTML, CSS e JavaScript puro.

O objetivo do projeto é criar experiências cinematográficas inspiradas em:

* Analog Horror
* ARGs
* VHS Horror
* Liminal Spaces
* Jogos de terror psicológico

Cada entidade possui:

* narrativa própria
* efeitos visuais únicos
* glitches personalizados
* ambientação diferente
* transições cinematográficas

---

# 🧩 Entidades

## 🌲 Slenderman

Atmosfera de paranoia e observação silenciosa.

## 🔪 Jason Voorhees

Experiência inspirada em filmes slasher clássicos.

## 🤖 Springtrap

Arquivo amaldiçoado com estética VHS corrompida.

## ⛏️ Herobrine

Lenda urbana digital dentro de um mundo Minecraft corrompido.

## 🎈 Pennywise

Terror psicológico com jumpscares, balões e glitches vermelhos.

## 🟨 Backrooms

Experiência liminal focada em desconforto psicológico e sensação de estar sendo observado.
