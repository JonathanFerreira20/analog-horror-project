🎥 ANALOG-HORROR-PROJECT

“Some tapes should never be found.”

Projeto de horror cinematográfico inspirado em creepypastas, jogos clássicos de terror, VHS analógico e ambientes perturbadores.
O objetivo é criar uma experiência visual extremamente imersiva misturando realismo com estética retro de gravações perdidas.

👁️ Sobre o Projeto

analog-horror-project é um projeto focado em criar artes, cenários, conceitos e futuras experiências interativas baseadas em horror analógico.

O projeto mistura:

📼 VHS Horror

🧟 Entidades do Projeto
👁️ Herobrine

Uma entidade misteriosa perdida dentro de mundos corrompidos do Minecraft.
Olhos completamente brancos, comportamento inexplicável e presença silenciosa em florestas cobertas por névoa.

Características:
Florestas corrompidas
Névoa pesada
Atmosfera paranormal
Animais com olhos brilhantes
Estruturas estranhas

Um assassino silencioso que aparece em áreas abandonadas e ambientes escuros.

Características:
Máscara destruída
Presença ameaçadora
Florestas noturnas
Terror slasher cinematográfico
Atmosfera brutal e silenciosa

Uma criatura alta e perturbadora escondida entre árvores e áreas isoladas.
